# Akan Name Generator

## Project Description
A web application that calculates the day of the week a user was born and assigns them an Akan name based on their gender. Akan names come from Ghanaian culture, where children are named based on the day they were born.

## Live Demo
https://github.com/CRANCKYY/Akan-name-generator/settings/pages

## Features
- Calculate day of the week from birthdate
- Generate authentic Ghanaian Akan names
- Input validation for dates and gender
- Responsive and user-friendly design
- Clean, modern interface with visual feedback

## Technologies Used
- **HTML5** - Page structure and semantics
- **CSS3** - Styling and responsive design
- **JavaScript (ES6)** - Core logic and interactivity
- **Git & GitHub** - Version control and deployment
- **GitHub Pages** - Free hosting
